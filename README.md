# Notes API — система управления личными заметками

## Описание

API для создания и управления личными заметками. Позволяет пользователям регистрироваться, создавать заметки с категориями, фильтровать и искать.

### Функционал
- Регистрация и аутентификация (Laravel Sanctum)
- CRUD заметок (создание, просмотр, редактирование, удаление)
- CRUD категорий
- Фильтрация заметок по категории и статусу
- Поиск по заголовку
- Пагинация

### Технологии
- PHP 8.x / Laravel 8.x
- MySQL
- Laravel Sanctum

---

## Установка и запуск

### Требования
- PHP >= 8.0
- Composer
- MySQL

### Шаги установки

```bash
# 1. Клонировать репозиторий
git clone [https://github.com/ваш-аккаунт/notes-api.git](https://github.com/Sasha1122323/notes-api.git )
cd notes-api

# 2. Установить зависимости
composer install

# 3. Создать .env файл
cp .env.example .env

# 4. Сгенерировать ключ приложения
php artisan key:generate

# 5. Настроить .env (укажите свои параметры)
#    DB_DATABASE=имя_базы
#    DB_USERNAME=пользователь
#    DB_PASSWORD=пароль

# 6. Выполнить миграции
php artisan migrate

# 7. Запустить сервер
php artisan serve
